# docs

專案內**參考文件與設計範例**的收納處，與 `.cursor` 規則搭配使用。

---

## 目錄

| 路徑 | 說明 |
|------|------|
| **ux-writing/** | UX writing 必讀文件（已收錄，路徑供 `.cursor/company-design-system/ux-writing.md` 參照） |
| **ux-writing/English/** | 英文：UX Writing Guideline、CSV、image |
| **ux-writing/日本語/** | 日文：UXライティングガイドライン、用語索引、CSV |
| **design-examples/** | 設計範例（與 company-design-system 對齊用） |
| **design-examples/Platform example/** | Platform 畫面範例（登入、Sidebar、Drawer、表格、Dialog 等） |
| **design-examples/Learner example/** | Learner 相關範例 |
| **REFERENCED-FILES.md** | 參考檔案清單與上傳 GitHub／路徑更新說明 |

路徑皆為**專案根目錄相對路徑**（如 `docs/ux-writing/English/...`），clone 後即可使用。
